                        1. INTRODUCCIÓN (1 minuto)
═══════════════════════════════════════════════════════════════════════════════

[MOSTRAR: Slide con título del proyecto o pantalla inicial]

LOCUTOR 1:
"Buenos días/tardes. Somos [Nombre del Grupo] y en esta presentación vamos a 
demostrar nuestro proyecto VitalWatch, un sistema de alertas médicas en tiempo 
real desarrollado con arquitectura Cloud Native.

VitalWatch es un sistema completo que permite a profesionales médicos monitorear 
signos vitales de pacientes críticos, generando alertas automáticas cuando los 
valores se encuentran fuera de rangos normales.

El proyecto ha sido desarrollado cumpliendo con todos los requisitos de la 
asignatura:
- Backend con Spring Boot y microservicios
- Frontend con Angular
- API Manager con Kong Gateway
- Autenticación y seguridad con Spring Security
- Base de datos Oracle Cloud
- Y completamente desplegado en Microsoft Azure

Comencemos mostrando la arquitectura general del sistema."

═══════════════════════════════════════════════════════════════════════════════
                   2. ARQUITECTURA DEL SISTEMA (1.5 minutos)
═══════════════════════════════════════════════════════════════════════════════

[MOSTRAR: Diagrama de arquitectura o archivo docs/arquitectura.md]

LOCUTOR 2:
"La arquitectura de VitalWatch está basada en microservicios y sigue el patrón 
Cloud Native. Permítanme explicar los componentes principales:

En la capa de presentación tenemos:
- El Frontend desarrollado en Angular 17, desplegado en Azure Container Apps
- Una interfaz responsive y moderna que se comunica con el backend mediante APIs REST

En la capa de aplicación contamos con:
- Kong Gateway como API Manager para gestión centralizada de endpoints
- El Backend BFF desarrollado con Spring Boot 3.2
- Implementamos Spring Security para autenticación y autorización
- Todos los microservicios están containerizados con Docker

Y en la capa de datos:
- Oracle Autonomous Database en Oracle Cloud Infrastructure
- Ubicado en la región de Santiago, Chile
- Conexión segura mediante Oracle Wallet con protocolo TLS/SSL

Todo el sistema está desplegado en Azure Container Apps, lo cual nos permite 
escalado automático y alta disponibilidad. Ahora les mostraré el despliegue en vivo."
